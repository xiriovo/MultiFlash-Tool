## 💡 使用建议

### 对于用户
1. **遇到 Bug**: 使用 Bug 报告模板提交 Issue
2. **功能建议**: 使用功能请求模板提交 Issue
3. **使用问题**: 先查看 FAQ，再到 Discussions 提问
4. **即时交流**: 加入 QQ 群或 Telegram 频道

### 对于贡献者
1. **代码贡献**: 提交 Pull Request
2. **文档改进**: 提交 Issue 或 PR
3. **问题讨论**: 参与 GitHub Discussions

### 对于维护者
1. 定期检查各平台的消息
2. 及时回复 Issues 和 PR
3. 维护社区氛围
4. 更新联系方式时同步所有文档
